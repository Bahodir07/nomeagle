# Badge images

Place badge PNGs here so the app can load them. The app expects these paths:

- **explorer/** — `first-step.locked.png`, `first-step.unlocked.png`, `cultural-traveler.locked.png`, `cultural-traveler.unlocked.png`
- **country/** — `kz-scholar.locked.png`, `kz-scholar.unlocked.png`
- **mastery/** — `perfect-quiz.locked.png`, `perfect-quiz.unlocked.png`
- **streak/** — `streak-3.locked.png`, `streak-3.unlocked.png`

If your images are in `src/assets/badges/`, copy the contents of each subfolder (explorer, country, mastery, streak) into this folder’s matching subfolder.
